# 📚 Library Management System

A modern **Library Management System** built with **Python, SQLite, and Streamlit**. The application allows librarians to efficiently manage books, members, book transactions, and reports through an interactive web interface.

---

## 🚀 Features

### 📊 Dashboard
- View total books, members, issued books, and overdue books.
- Interactive charts using Plotly.
- Recent transaction history.

### 📚 Book Management
- Add new books.
- View all books.
- Search books by Title, Author, Category, or ISBN.
- Update book details.
- Delete books.

### 👤 Member Management
- Register new members.
- View all members.
- Search members.
- Update member information.
- Delete members.

### 🔄 Transaction Management
- Issue books to members.
- Return books.
- Track issue and due dates.
- View complete transaction history.

### 📈 Reports
- Library summary.
- Overdue books report.
- Transaction report.
- Download reports as CSV.

### ⚙ Settings
- Application information.
- Database status.
- Project information.

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/LibraryManagementSystem.git
```

### Move into Project Folder

```bash
cd LibraryManagementSystem
```

### Create Virtual Environment

Windows

```bash
python -m venv venv
```

Activate

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
